# KT Inventory v1.0.0

Initial public release of the `KT Inventory` module for Perfex CRM.

## Added

- Warehouse management with manager assignment and active status control
- Inventory item management with SKU, min/max stock, and activation state
- Multi-barcode support with primary barcode selection and generated internal barcodes
- GS1 parsing for lot, expiry date, and serial number extraction
- Goods receipt, goods issue, stock adjustment, and stock transfer flows
- Reservations, stock balances, transaction history, and dashboard summary
- Batch, lot, expiry, and recall trace reporting
- CSV and Excel export tools plus CSV import templates
- Invoice reservation hooks for create, update, cancel, and delete events
- Install and uninstall scripts for schema and module options

## Security

- Capability-based access control for module menus, actions, APIs, and barcode scan endpoints
